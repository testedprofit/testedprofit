Features

Airdrops, payrolls, and distributions — simplified

Whether you're running an airdrop or paying contributors, Multisend handles it in one shot.

CSV Paste Import

Paste your recipient list in ADDRESS,AMOUNT format. Handles hundreds of rows instantly — no UI row-by-row entry needed for large drops.

ALGO or Any ASA

Leave the Asset ID blank to send ALGO, or enter any ASA ID to distribute your token. One tool for every Algorand asset type — no switching needed.

Atomic Transaction Group

All sends are bundled in a single Algorand transaction group. They are atomically linked — if any send fails, none go through. Zero partial distributions.

Process

How it works

1

Connect your wallet

Open AlgoFlow Multisend and connect via Pera Wallet. Use TestNet to run a test distribution with no real assets.

2

Choose your asset

Leave Asset ID blank to send ALGO, or paste any ASA ID to distribute that token to your recipients.

3

Add recipients

Paste your CSV (ADDRESS,AMOUNT) or add rows manually. The tool validates every address before proceeding.

4

Sign the batch transaction

Review the full distribution in Pera Wallet and sign once. All sends are submitted atomically — done in a single Algorand block.
