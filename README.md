## 🚀 AlgoFlow — my main project

**[AlgoFlow](https://testedprofit.com/pages/algoflow/)** is a 12-tool **Algorand Web3 suite, live on mainnet**. There's no backend — every action is a real, signed, on-chain transaction.

| Tool | What it does | Status |
|------|--------------|--------|
| **[PNETLaunchPad](https://testedprofit.com/pages/genesis/)** | Bonding-curve token launchpad — tiered launch fees, automatic graduation to a Tinyman v2 liquidity pool, LP burn | 🟢 Mainnet |
| **[AlgoPool](https://testedprofit.com/pages/algopool/)** | No-loss / prize-linked savings pool | 🟢 Mainnet |
| **[ProfitLock](https://testedprofit.com/pages/profitlock/)** | Trustless token time-locks | 🟢 Mainnet |
| **Crowdfund** | On-chain fundraising campaigns | 🟡 Testnet |
| MultiSender · Wallet Inspector · Token Launchpad · AlgoMarkets · AlgoLearn · AlgoSocial · ShapeMap | Free Algorand utilities | 🟢 Live |

**Stack:** React + Vite + Tailwind on the frontend · **PuyaPy / TEAL** smart contracts · `algosdk` for signing real mainnet transactions.
